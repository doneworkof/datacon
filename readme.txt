Данный репозиторий включает в себя решения
трёх тасков. Они распределены по папкам:

1. task1
2. task2
3. task3

В каждой из этих папок находится свой файл readme.txt,
который поясняет значение главных файлов решения.

Также в каждой из папок находится файл формата .docx,
описывающий задание.

В файле requirements.txt вы можете найти
список всех необходимых библиотек и их версий
для запуска каждого файла .ipynb или .py из этого
репозитория.
